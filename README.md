# Interview Dojo

## About Interview Dojo

Interview Dojo is a platform designed to assist candidates in preparing for their job interviews. It provides tools and resources tailored to enhance the interview process.

## Features
- **Mock Interviews:** Practice interviews that simulate real interview scenarios.
- **Feedback Mechanism:** Get personalized feedback from experts after mock interviews.
- **Resource Library:** Access to a library of interview questions, tips, and articles.
- **Scheduling:** Ease of scheduling interviews with experts at your convenience.

## Stack
- **Frontend:** React.js
- **Backend:** Node.js with Express
- **Database:** MongoDB
- **Deployment:** Heroku

## Quick Start Guide
1. Clone the repository: `git clone https://github.com/nkosikhumalo/Interview-Dojo.git`
2. Navigate into the project directory: `cd Interview-Dojo`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
